# Learning Guide

## 프로젝트 성격

이 프로젝트는 배포 중심이 아니라 기능 구현 연습과 비교 학습이 목적입니다.

## 추천 학습 순서 체크리스트

- [ ] Web 앱 실행 후 화면 구조와 상태 흐름 파악
- [ ] HSL -> RGB -> HEX 계산식 코드 추적
- [ ] SwiftUI 포트에서 동일 기능 구현 방식 비교
- [ ] 저장/히스토리 동작 차이 정리
- [ ] 기능 체크리스트로 회귀 확인

## 실습 과제

### 과제 A: 계산 로직 이해

- [ ] Web의 `hslToRgb`, `rgbToHex` 직접 테스트
- [ ] Swift Color 유틸 구현과 결과 비교
- [ ] 동일 입력에서 HEX 결과 일치 검증

### 과제 B: 기능 parity 맞추기

- [ ] Web에서 히스토리 클릭 시 HSL 복원 구현
- [ ] Web에 Save Collection(localStorage) 추가
- [ ] Swift와 같은 사용자 시나리오로 동작 비교

### 과제 C: 리팩터링

- [ ] 계산 로직을 별도 유틸 파일로 분리
- [ ] UI 컴포넌트 분리(슬라이더, 팔레트, 히스토리)
- [ ] 변경 전후 체크리스트 기준으로 회귀 테스트

## 검증 포인트 체크리스트

- [ ] 슬라이더 변경 후 색상이 즉시 반영되는가
- [ ] HEX가 항상 7자리 형식(`#RRGGBB`)을 유지하는가
- [ ] 히스토리 선택 시 기대 상태로 복원되는가
- [ ] 저장/재실행 후 데이터가 유지되는가
